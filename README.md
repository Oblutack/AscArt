# AscArt

A Cyberpunk-themed Desktop ASCII Art Generator.

## Tech Stack
- **Frontend**: React.js (UI), Electron (Desktop wrapper)
- **Backend**: Python 3.12 (Image processing engine)
- **Libraries**: Pillow, Numpy, RemBG, OpenCV

## Requirements
- Python 3.12+
- Node.js & npm

## Setup
1. Install dependencies: `npm install`
2. Setup Python venv: 
   - `cd python`
   - `python -m venv venv`
   - `pip install -r requirements.txt`
